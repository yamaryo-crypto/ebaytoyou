# ツール販売時の利用規約メモ（Streamlit・eBay）

※ これは規約の要点整理であり、法的助言ではありません。販売前に必ず公式の利用規約を最新版でご確認ください。

---

## 1. Streamlit（Community Cloud とライブラリ）

### ライブラリ（Streamlit 本体）
- **Apache 2.0 ライセンス**で提供されており、**商用利用・有料アプリの開発・販売は可能**です。
- 出典: [Streamlit Terms of Use](https://streamlit.io/deployment-terms-of-use)（「The Software is not governed by the below terms」）

### Community Cloud（無料ホスティング）
- **個人・非商用目的**に限るとの制限があります。
- 規約の要約より:
  - 「Community Cloud で**個人を特定できる情報（PII）を処理する**アプリは、**個人利用および非商用目的**にのみ利用可能」
  - 「Community Cloud 上のアプリで、**金融・医療・生体情報その他の機密個人情報を処理してはならない**」
- **Addendum（Community Cloud Content）** より:
  - 個人情報・PII 等を処理する利用は、**個人・非商用**（評価・教育・家庭内利用など）に限られ、
  - **業務・商用目的**で Community Cloud を利用してはならない、と明記されています。

**販売する場合の目安:**
- **Streamlit Community Cloud 上で「有料で提供するサービス」として動かす**のは、規約上 **商用利用** に当たり、**推奨されません**。無料の「個人用」としての利用が前提です。
- **販売するツール**として運用するなら、**自前サーバーや VPS、Streamlit の有料ホスティング（Snowflake 等）** など、Community Cloud 以外のホストにデプロイすることを検討してください。
- アプリの**ソースコードやパッケージを販売**し、お客様が**ご自身の環境で動かす**形であれば、Streamlit ライブラリの利用として **Apache 2.0 の範囲内で問題ありません**。

## 2. eBay Developers Program / API

- 出典: [eBay Developers Program – Terms of Use and API License Agreement](https://developer.ebay.com/join/api-license-agreement)

### 目的
- プログラムと Developer Tools は、**eBay サービスへのアクセスと利用を促進・支援する目的**で提供されています。
- **eBay のビジネス利益を損なうような利用**と判断された場合、**API アクセスの停止・プログラム終了**などの措置が取られ得ます。

### 許可される利用（Authorized Use）
- 自分自身または**利用者（Your Users）**が eBay サービスを利用することを**促進する**アプリケーションの開発・運用が想定されています。
- eBay Content の表示・改変は、規約で認められた範囲で行います。

### 再許諾（サブライセンス）
- **利用者（Your Users）** に、**あなたのアプリを通じて** eBay Content を表示する権利を許諾することは可能です。
- 条件として:
  - 利用者と**拘束力のある契約**を結び、**eBay の API ライセンス契約に従う**旨を盛り込む。
  - **API 呼び出しはあなたの枠にカウント**される。
  - **Application Keys を利用者に開示・共有してはならない**（あなたのアプリ経由でのみ API を叩く形にする）。

### 禁止行為の例（RESTRICTED ACTIVITIES）
- eBay Content を**販売・賃貸・トレード・配布・リース**などで**商用化**すること（規約で認められた利用を除く）。
- eBay サービスと**競合**するサイト・サービスを**設計・構築・宣伝・拡張**すること。
- API の**過剰・不正利用**、eBay のポリシー・法令違反を**助長するアプリ**の設計・提供。

### 販売する場合の目安
- **「eBay のデータ（eBay Content）そのものを販売・転売する」** ことは禁止されています。
- **「eBay API を利用したアプリケーション（ツール）そのものを販売する」** ことは、規約上**明文で禁止されていません**。  
  - 利用者が自分の eBay アカウントで OAuth 等により認証し、**あなたのアプリが許可された目的（例: 出品画像の盗用検知）で API を利用する**形であれば、**Authorized Use の範囲**に含まれると考えられます。
- 販売する場合は:
  - **利用者との利用規約**で、eBay の利用規約・API ライセンス契約の遵守を明記する。
  - **Application Keys は開示せず**、あなたのアプリケーション経由でのみ API を呼び出す。
  - **利用量・用途**が、eBay の「合理的なリクエスト量」やドキュメントのガイドラインに反しないようにする。

## 3. まとめ（販売時に気をつけること）

| 項目 | 目安 |
|------|------|
| **Streamlit Community Cloud** | 無料・個人・非商用向け。**有料サービスとしての運用は規約上不向き**。販売用なら自前ホストや有料ホスティングを検討。 |

| **共通** | 規約は改定されるため、**販売前・運用中に公式の最新 Terms を必ず確認**し、必要なら法務・弁護士に相談してください。 |
